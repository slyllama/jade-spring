### Decorations
/cleardeco		Clear all decorations
/loaddeco		Load decorations from file
/resetdeco		Reset decorations for defaults
/savedeco		Save decorations to file

### Visibility
/time=night		Switch to night
/time=day		Switch to day
/showplayer		Show the player mesh
/hideplayer		Hide the player mesh
/showcrumbs		Show bugs, Dragonvoid etc
/hidecrumbs		Hide bugs, Dragonvoid etc

### Generic utilities
/addkarma=X		Add X Karma (negatives permitted)
/collision=off	Disable player collisions
/collision=on	Enable player collisions
/giftletter		Test out the gift letter
/gravity		Toggle jumping puzzle physics
/karmastats		Print Karma statistics
/printsavedata	Print all save data
/printsettings	Print user settings
/savedata		Save data to file
/screenshot		Take a screenshot
/speedratio=X	Set the player's speed to ratio X of its base speed
/togglebook		Toggle compendium visibility

/enableattn		Enable Attenuator (good for if story hasn't spawned it yet)
/disableattn	Disable Attenuator
